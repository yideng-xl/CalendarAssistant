- **地点/链接**: https://365.kdocs.cn/l/crrVk6GwcCV8
- **提醒**: 15分钟前、5分钟前、开始时 (Sound Alarm)

## 3. 实现方案 (方案 B - 方式 1)
编写一个独立的 Python 脚本 `CalendarAssistant/sync_v1_19_4.py`：
- 复用 `calendar_sync_prototype.py` 中的 `sync_to_mac_calendar` 核心逻辑。
- 硬编码上述会议参数。
- 运行脚本完成同步。

## 4. 验证计划
- 运行脚本。
- 检查输出是否为 `SUCCESS`。
- 打开 macOS 日历确认日程及 3 个提醒设置是否正确。
