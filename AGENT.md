# Agent Configuration & Preferences

## 核心指令 (Core Directives)
- **语言偏好**：所有交流、文档、代码注释及反馈必须使用 **中文**。
- **时区设定**：默认地区为 **上海**，所有时间显示与逻辑处理均采用 **北京时间 (CST/UTC+8)**。

## 工作模式 (Working Mode)
- 遵循 `Superpowers` 框架。
- 在进行任何重大功能开发或重构前，必须先激活 `brainstorming` 技能进行设计确认。
- 严格执行 **测试驱动开发 (TDD)**，在编写实现代码前先编写测试用例。

## 项目背景 (Project Context)
- **CalendarAssistant**：系统级日程同步助手（Tray App），旨在解决协同工具碎片化导致的日程同步难题。
- **技术栈演进**：当前为 Python MVP，计划向 Go 语言重构。
